# HollowKit

光造形(SLA/DLP)プリンターでの出力向けに、Blender のモデルを **中空化**し、
レジンの排出・エア抜き用の **穴あけ**を行う Blender 5.0 アドオン(拡張機能)。

処理はすべて **1 つの Geometry Nodes モディファイア**で行うため **非破壊**で、
壁厚や穴の位置をいつでも調整でき、最後に「確定」して確定メッシュに焼き込めます。

## なぜ中空化・穴あけが必要か

光造形では造形物が中実だとレジンを大量に消費し、硬化収縮で反り・割れが
起きやすくなります。内部を殻(シェル)状に中空化し、最下部などに排出穴を
開けておくと、未硬化レジンが抜けて内部に溜まらず(＝バキューム効果による
プレート剥がれや破裂も防げる)、材料も節約できます。

## 仕組み(すべて Geometry Nodes)

```
【① 中空化グループ】
ジオメトリ → SDF化 → 内側へ壁厚ぶんオフセット → 空洞面をメッシュ化
  → 小空洞を選別 → (キャッシュ切替) → 軸柱を差し引き → 面反転
  → 元ジオメトリと結合 →「中空化を確定」で焼き込み

【② 穴あけグループ(独立モディファイア)】
確定済みシェル → 穴マーカーの位置から矢印方向へ直方体を差し引き
  →「穴あけを確定」で焼き込み
```

* **中空化**: メッシュを SDF グリッドに変換し、壁厚ぶん内側にオフセットした
  グリッドから**内側の空洞面だけ**をメッシュ化。面を反転して元メッシュと
  結合し殻にする。**外側は元メッシュをそのまま使うため、元のディテールは
  一切変わらない**。解像度(ボクセル)は空洞面の細かさにのみ影響する。
* **空洞の選別**: 空洞が複数の閉じた空間に分かれた場合、小さい空洞は
  レジン溜まりになるため作らず中実のまま残す(アイランドごとに符号付き
  体積を合算して判定)。モードは 3 つ:
  * **最大のみ**(既定) — 最も大きい空洞だけを残す。指の間・髪の隙間などの
    小空洞を調整不要で確実に排除できる。
  * **サイズ指定** — 「最小空洞径」の球より大きい空洞をすべて残す。
    複数パーツを結合したメッシュで大きい空洞が複数あるときに。
  * **全て残す** — 選別しない。
* **軸打ち(中実柱)**: 「軸マーカー」(矢印の Empty)を置いた位置から矢印方向へ
  指定した径・長さの中実柱を残す(空洞から柱を差し引く)。ダボ面などに置けば
  中空化しても軸(真鍮線など)を打てる。
* **穴あけ**: 「穴マーカー」(矢印の Empty)の位置から**矢印の方向へ**
  直方体(8頂点、ブーリアンが軽い)をブーリアン差し引きして貫通させ、
  中空部を外へ通じさせる。軸柱も同様にマーカーの矢印方向へ伸びる四角柱。
* **段階分けによる軽量化**: 「中空化を適用」時に重い空洞計算(SDF)を一度だけ
  行って自動的に固定(キャッシュ)する。以後の軸柱・穴あけは軽いブーリアン
  だけで再計算されるため、マーカーの調整が軽い。壁厚など中空化の設定を
  変えると固定は解除されるので、調整後に「中空化を更新」で再固定する。
* **高速ブーリアン(自動フォールバック)**: 穴あけはまず Manifold ソルバー
  (高速・水密。EXACT は高密度メッシュで非多様体を出す)で切り、モデルが
  水密でなく結果が空になった場合は自動で EXACT に切り替わる。マーカーが
  0 個のときはブーリアン自体をスキップするので、メッシュが消えることはない。
* **空洞プレビュー**: 空洞と軸柱の形をワイヤフレームで重ね表示し、
  マーカーを動かしながらリアルタイムで確認できる。

## 使い方(2 段階ワークフロー)

処理は「① 中空化」「② 穴あけ」の 2 段階に分かれ、それぞれ独立した
Geometry Nodes モディファイアで行う。段階を分けることで、重い空洞計算を
繰り返さずに済む。

### ① 中空化 + 軸打ち

1. サイドバー(N キー) > **HollowKit** タブを開く。
2. 対象メッシュを選択して **「中空化を適用」**。モディファイアが付き、
   計算結果が自動で固定(キャッシュ)される。
3. **壁厚**・**解像度**・**空洞の選別**を調整(変更すると固定が解除される
   ので、調整後に「中空化を更新」で再固定)。
4. 軸打ちしたい場所(ダボ面など)に **「軸マーカーを追加」**。マーカーから
   矢印方向へ中実の四角柱が残る。**「空洞プレビュー」**で空洞と柱をワイヤ
   表示しながら調整できる。
5. **「中空化を確定」**でメッシュへ焼き込み、段階①を終える。

### ② 穴あけ

6. **「穴マーカーを追加」**でビュー上の面をクリックすると矢印マーカーが
   置かれ、**矢印の方向へ**掘られる。初回追加時に穴あけモディファイアが
   自動で付く。確定済みシェルへの単独ブーリアンなので軽い。
   * 「穴をライブ表示」を切ればマーカー調整は完全にゼロコスト。
7. **穴の幅**(2〜4mm 目安)を設定。長さ「自動」は壁を貫いて空洞に届くだけ
   (反対側は貫通しない)。
8. **「穴あけを確定」**で焼き込み完了。エクスポートへ。
   * やり直したいときは **「解除」**でモディファイアとマーカーを消して元に戻る。

## パラメータ

| 区分 | 項目 | 説明 |
|------|------|------|
| 中空化 | 壁厚 | シェルの厚み。光造形では 1.5〜3mm 目安 |
|        | 解像度(自動/手動) | 空洞面の細かさ。壁厚より十分小さく。外側には影響しない |
|        | 滑らかさ(面削減) | 空洞面の平坦部の面をまとめてポリゴン削減(0=最忠実) |
|        | 空洞の選別 | 最大のみ(既定)/サイズ指定/全て残す。レジン溜まり防止 |
|        | 最小空洞径 | サイズ指定モード時: この径の球より小さい閉空洞は埋める |
| 軸打ち | 軸柱の幅 | マーカー位置に残す中実四角柱の幅。3mm 軸なら 8〜12mm 目安 |
|        | 軸柱の長さ | マーカーから矢印方向へ残す中身の長さ |
| 穴あけ | 穴の幅 | 排出/エア抜きの四角穴の幅(一辺)。2〜4mm 目安 |
|        | 穴の長さ(自動/手動) | 自動=中空時は壁を貫いて空洞に届くだけ(反対側は貫通しない)。非中空時は全体貫通 |

## インストール

Blender 5.0 で `dist/hollowkit-0.1.0.zip` を
**編集 > プリファレンス > アドオン > 右上の ▽ > ディスクからインストール**
で導入する。

## 開発 / テスト

ヘッドレステスト(他アドオンの干渉を避けるため `--factory-startup` 必須):

```
& "C:\Program Files\Blender Foundation\Blender 5.0\blender.exe" \
    -b --factory-startup --python _test/test_pipeline.py
```

拡張機能のビルド:

```
& "C:\Program Files\Blender Foundation\Blender 5.0\blender.exe" \
    --command extension build --source-dir . --output-dir ./dist
```

## 動作環境

* Blender **5.0** 以上(SDF グリッドノード `Mesh to SDF Grid` / `SDF Grid Offset`
  / `Grid to Mesh` を使用するため)。
* シーンの単位を実際の造形サイズ(mm など)に合わせておくと、壁厚・穴径を
  実寸で指定できる。
