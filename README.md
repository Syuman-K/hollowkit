# HollowKit

光造形(SLA/DLP)プリンターでの出力向けに、Blender のモデルを **中空化**し、
レジンの排出・エア抜き用の **穴あけ**を行う Blender 5.0 アドオン(拡張機能)。

処理はすべて **1 つの Geometry Nodes モディファイア**で行うため **非破壊**で、
壁厚や穴の位置をいつでも調整でき、最後に「確定」して確定メッシュに焼き込めます。

## なぜ中空化・穴あけが必要か

光造形では造形物が中実だとレジンを大量に消費し、硬化収縮で反り・割れが
起きやすくなります。内部を殻(シェル)状に中空化し、最下部などに排出穴を
開けておくと、未硬化レジンが抜けて内部に溜まらず(＝バキューム効果による
プレート剥がれや破裂も防げる)、材料も節約できます。

## 仕組み(すべて Geometry Nodes)

```
ジオメトリ ─┬─(中空化)→ SDF化 → 内側へ壁厚ぶんオフセット → 空洞面をメッシュ化
            │           → 小空洞を削除 → 面反転 → 元ジオメトリと結合 ─┐
            └──────────────────────────────────────────────────────────┴─→ ベース
ベース ─┬─(穴あけ)→ 穴マーカー(コレクション) → 位置・向きにシリンダーを差し引き ─┐
        └──────────────────────────────────────────────────────────────────────────┴─→ 出力
```

* **中空化**: メッシュを SDF グリッドに変換し、壁厚ぶん内側にオフセットした
  グリッドから**内側の空洞面だけ**をメッシュ化。面を反転して元メッシュと
  結合し殻にする。**外側は元メッシュをそのまま使うため、元のディテールは
  一切変わらない**。解像度(ボクセル)は空洞面の細かさにのみ影響する。
* **空洞の選別**: 空洞が複数の閉じた空間に分かれた場合、小さい空洞は
  レジン溜まりになるため作らず中実のまま残す(アイランドごとに符号付き
  体積を合算して判定)。モードは 3 つ:
  * **最大のみ**(既定) — 最も大きい空洞だけを残す。指の間・髪の隙間などの
    小空洞を調整不要で確実に排除できる。
  * **サイズ指定** — 「最小空洞径」の球より大きい空洞をすべて残す。
    複数パーツを結合したメッシュで大きい空洞が複数あるときに。
  * **全て残す** — 選別しない。
* **穴あけ**: 「穴マーカー」(矢印の Empty)を置いた位置・向きにシリンダーを
  ブーリアン差し引きして貫通させ、中空部を外へ通じさせる。

## 使い方

1. サイドバー(N キー) > **HollowKit** タブを開く。
2. 対象メッシュを選択して **「中空化を適用」**。Geometry Nodes モディファイアが
   付き、以後パラメータ変更は即プレビューに反映される。
3. **壁厚**・**解像度(ボクセル)**を調整。解像度は「自動」(最大寸法 / ディテール)
   または「手動」(絶対ボクセルサイズ)。細かいほど忠実だが重くなる。
4. **「穴マーカーを追加」**でビュー上の面をクリックすると、その表面に矢印
   マーカーが置かれ、矢印がモデル内側(＝掘る方向)を向く。移動・回転で調整可。
   複数置ける。排出穴は最下部に下向きで置くのが基本。
   * クリック位置が取れない場合は 3D カーソル位置に追加される。
     `Shift + 右クリック`で 3D カーソルを表面にスナップしておくと便利。
5. **穴の径**(2〜4mm 目安)を設定。穴の長さは「自動」で確実に貫通する。
6. 仕上げに **「確定」**でモディファイアを適用して確定メッシュにする
   (マーカーは削除される)。エクスポート前に実行。
   * やり直したいときは **「解除」**でモディファイアとマーカーを消して元に戻る。

## パラメータ

| 区分 | 項目 | 説明 |
|------|------|------|
| 中空化 | 壁厚 | シェルの厚み。光造形では 1.5〜3mm 目安 |
|        | 解像度(自動/手動) | 空洞面の細かさ。壁厚より十分小さく。外側には影響しない |
|        | 滑らかさ(面削減) | 空洞面の平坦部の面をまとめてポリゴン削減(0=最忠実) |
|        | 空洞の選別 | 最大のみ(既定)/サイズ指定/全て残す。レジン溜まり防止 |
|        | 最小空洞径 | サイズ指定モード時: この径の球より小さい閉空洞は埋める |
| 穴あけ | 穴の径 | 排出/エア抜き穴の直径。2〜4mm 目安 |
|        | 穴の長さ(自動/手動) | ドリルの長さ。自動=最大寸法×2 で確実に貫通 |

## インストール

Blender 5.0 で `dist/hollowkit-0.1.0.zip` を
**編集 > プリファレンス > アドオン > 右上の ▽ > ディスクからインストール**
で導入する。

## 開発 / テスト

ヘッドレステスト(他アドオンの干渉を避けるため `--factory-startup` 必須):

```
& "C:\Program Files\Blender Foundation\Blender 5.0\blender.exe" \
    -b --factory-startup --python _test/test_pipeline.py
```

拡張機能のビルド:

```
& "C:\Program Files\Blender Foundation\Blender 5.0\blender.exe" \
    --command extension build --source-dir . --output-dir ./dist
```

## 動作環境

* Blender **5.0** 以上(SDF グリッドノード `Mesh to SDF Grid` / `SDF Grid Offset`
  / `Grid to Mesh` を使用するため)。
* シーンの単位を実際の造形サイズ(mm など)に合わせておくと、壁厚・穴径を
  実寸で指定できる。
